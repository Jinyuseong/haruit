# Haruit

하루하루 미션을 수행하며 경험치를 쌓고, 레벨업하는 습관 관리 앱입니다.  
클라이언트와 서버를 모두 포함한 모노레포 구조로 개발되었습니다.

## 📌 주요 기능
- 미션 생성, 수정, 삭제
- 경험치 및 레벨 시스템
- 백엔드(NestJS)와 연동된 데이터 관리
- 프론트엔드(React) UI 구성

## 🛠 기술 스택
- **Frontend:** React, TypeScript
- **Backend:** NestJS, TypeScript
- **Database:** (추가 예정)
- **버전 관리:** Git & GitHub
